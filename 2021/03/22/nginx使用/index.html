location匹配规则

  • 空格：默认匹配，普通匹配

  • =：精确匹配

  • ~*：匹配正则表达式，不区分大小写

  • ~：匹配正则表达式，区分大小写

  • ^~：以某个字符路径开头

nginx.pid打开失败以及失效的解决方案

cd /var/run/nginx/nginx.pid 会出现 没有那个目录或文件

  1. 重新创建了这个文件，使用如下命令：mkdir /var/run/nginx/

  2. cd /var/run/nginx

  3. cd /usr/local/nginx

  4. cd sbin

  5. ./nginx -t

  6. ./nginx -s reload 会报PID错误，nginx.pid是无效的。

  7. ./nginx -c /usr/local/nginx/conf/nginx.conf 重新设置nginx.conf默认配置文件

  8. ./nginx -s reload 重启nginx
