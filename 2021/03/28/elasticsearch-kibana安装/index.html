发表于 2021-03-28 | 分类于 elastic - elasticsearch安装 - kibana安装

上传后解压

解压后将文件移至/usr/local/下

进入/usr/local/elasticsearch-7.12.0

ES目录介绍

  • bin : 可执行文件，运行es的命令，包含一些脚本文件
  • config ： 配置文件
  • JDK ：java环境
  • lib：依赖的jar，类库
  • logs：日志文件
  • modules：es相关的模块
  • plugins：插件
  • data : 需自行创建的目录，作为索引目录mkdir data

进入config文件修改配置文件elasticsearch.yml

  • cluster.name : 集群名称

cluster.name: my-elasticsearch-1

  • node.name ：节点名称

node.name: es-node0

  • path.data ：data数据保存地址

path.data: /usr/local/elasticsearch-7.12.0/data

  • path.logs : 修改日志保存地址

path.logs: /usr/local/elasticsearch-7.12.0/logs

  • network.host ：绑定es网络IP

network.host: 0.0.0.0

  • cluster.initial_master_nodes ：集群节点，名字可以改成之前那个节点名称

cluster.initial_master_nodes: ["es-node0"]
