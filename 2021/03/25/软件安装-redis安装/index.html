目的要把redis作为开机自启

创建/usr/local/redis

用于存放配置文件

回redis解压文件拷贝配置文件

修改核心配置文件redis.conf

  1. daeminize no -> daeminize yes 目的：为了让redis启动在linux后台运行
  2. dir ./->dir /usr/local/redis/working 目的：修改redis工作目录
  3. bind 127.0.0.1 -::1->bind 0.0.0.0 目的：可以让远程连接，不收IP限制
  4. # requirepass foobared->requirepass myredis123

修改redis_init_script

CONF="/etc/redis/${REDISPORT}.conf"->CONF="/usr/local/redis/redis.conf"

为redis_init_script赋权

运行脚本启动redis
